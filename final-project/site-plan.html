Penny Project Site Plan

Mia Smith

WDD 131-02

Overview

Purpose

The purpose of this website is simply all about pennies. It will provide a brief overview of the history of pennies as well as have a page where users can keep track of the pennies they have.

Audience

Our audience is anyone interested in pennies.

Dynamic elements

I would like to make to counter page use a good chunk of javascript. Basically, I'd like users to be able to input what type and how many pennies they have. I'd like to add a search bar to look for pennies by year as well. I'd also like the home page to cycle through displaying random facts.

Content

Home page

Will randomly cycle through a series of fun facts and explain the purpose of the site, as well as link to the content pages.

History

This page will include a brief description and images of each of the following different kinds of pennies:
1. Fugio cent
2. Flowing Hair Chain
3. Flowing Hair Wreath
4. Liberty Cap
5. Draped Bust
6. Classic Head
7. Coronet
8. Braided Hair
9. Flying Eagle Cent
10. Indian Head Cent
11. Lincoln Cent

Counter

This page will allow users to input and keep track of any pennies that they have. There will be search and filter options to organize and sort through the information. Users will input the year of the penny, and then the website should automatically assign it to the correct design category.
